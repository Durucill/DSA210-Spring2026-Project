# Netflix Hours vs Snack Consumption

## Dataset
This dataset shows the relationship between Netflix watching hours and snack consumption.

## Variables
- netflix_hours: Number of hours spent watching Netflix
- snack_score: Level of snack consumption (1–5 scale)

## Analysis
A scatter plot was created to analyze the relationship between Netflix usage and snack consumption.

## Conclusion
There appears to be a positive relationship: as Netflix hours increase, snack consumption also tends to increase.
The dataset is small, so conclusions are indicative rather than definitive.
## Hypothesis Test

H0 (Null Hypothesis): There is no relationship between Netflix hours and snack consumption.  
H1 (Alternative Hypothesis): There is a relationship between Netflix hours and snack consumption.

The correlation coefficient between Netflix hours and snack consumption is approximately 0.97, indicating a very strong positive relationship.

Therefore, we reject the null hypothesis and conclude that there is a significant relationship between the variables.
